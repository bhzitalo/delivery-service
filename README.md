# 🧹 Delivery Servicer - Nosso Server de Minecraft

Servidor privado de Minecraft criado para jogar com amigos, focado em **survival**, exploração e diversão em grupo.

> ⚠️ Este servidor é particular e destinado apenas a amigos e convidados.

## 📌 Sobre o Servidor

Aqui valorizamos:
- 🤝 Cooperação entre jogadores
- 🏗️ Construções criativas
- ⚔️ Exploração e aventura
- 🎮 Jogabilidade equilibrada com mods

O servidor utiliza **Fabric + Modpack personalizado**, trazendo melhorias de qualidade de vida sem fugir da essência do Minecraft.

## ⚙️ Requisitos e Instalação

### 📋 Requisitos
- Minecraft Java Edition 
- Versão: 1.21.1
- Fabric Loader instalado [download](https://fabricmc.net/use/installer/)
- Modpack do servidor [download](https://www.mediafire.com/file/eeg34ypdvesk0mw/delivery-service-modpack.rar/file)

### 🛠️ Como instalar

1. Instale o **Minecraft Java Edition**
2. Instale o **Fabric Loader**
3. Baixe o modpack
4. Extraia os arquivos
5. Coloque a pasta `mods` dentro da `.minecraft`
7. Inicie o jogo com o perfil **Fabric**
