## 🛠️ Instalação

1. Instale o **Minecraft Java Edition 1.21.11**
2. Instale o [**Fabric Loader**](https://fabricmc.net/use/installer/)
3. Baixe o **Modpack**
4. Extraia os arquivos
5. Coloque a pasta `mods` dentro da `.minecraft`
6. Inicie o jogo com o perfil **Fabric**

```bash
.minecraft/
├── mods/           # Adicione aqui todos os mods do modpack (obrigatório)
├── resourcepacks/  # Pacotes de textura para melhorar a aparência do jogo (opcional)
└── shaderpacks/    # Shaders para efeitos visuais avançados (opcional)
```
